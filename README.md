# AI_Talk - 虚拟角色聊天系统

一个基于Web的虚拟角色聊天平台，支持三种心情风格，用户可以创建自定义AI角色，进行单聊或群聊，所有数据本地存储。

---

## 🌟 特色功能

### 三种心情风格
每个风格都有独立的界面设计、角色和记忆数据：

| 风格 | 说明 | 适用场景 |
|------|------|----------|
| ☀️ **热情阳光** | 暖橙色调，明亮活泼 | 好心情、热恋期、分享开心事 |
| 🌙 **Emo时刻** | 蓝紫色调，温柔治愈 | 坏心情、失恋、科研不顺利 |
| ⛓️ **暗黑深渊** | 纯黑深红，冷酷尖锐 | 心情极其差时的归宿 |

## 📖 使用指南

### 1. 选择心情风格
- 访问主页后，点击对应的风格卡片进入
- 每种风格有完全独立的角色和聊天记录

### 2. 角色管理

#### 创建角色
1. 点击左侧「🎭 角色」标签
2. 点击「+ 创建新角色」
3. 填写角色信息：
   - **角色名称**（必填）：角色的唯一标识
   - **选择模型**：在 config.yaml 中配置
   - **角色人设**（必填）：描述角色的背景、性格特点、说话风格等
   - **历史记忆**：角色的背景知识（可通过文件上传）
   - **上传文件**：支持 .md, .docx, .pdf, .xlsx 格式

#### 记忆总结
- 点击角色卡片上的「📝」按钮
- 系统会自动总结该角色的所有聊天记录
- 总结结果会追加到角色的历史记忆中

### 3. 聊天功能

#### 单聊
- 在角色列表中点击任意角色即可开始聊天
- 消息支持 Markdown 渲染
- 支持发送图片（会自动识别图片内容）

#### 群聊
1. 点击左侧「👥 群聊」标签
2. 点击「+ 创建群聊」
3. 填写群聊名称并选择至少2个角色
4. 开始群聊，所有角色会依次回复

### 4. 聊天记录管理

#### 查看历史会话
- 点击左侧「💬 记录」标签
- 可切换查看单聊/群聊记录
- 点击任意记录可继续聊天

#### 删除会话
- 点击会话卡片上的「×」按钮可删除

### 5. 切换风格
- 点击聊天界面左上角的「←」按钮返回主页
- 重新选择其他心情风格

## ⚙️ 配置说明

### 模型配置 (config.yaml)
支持配置多个 OpenAI 兼容格式的大模型：

```yaml
llm_models:
  - name: "默认模型"
    base_url: "https://api.example.com/v1"
    api_key: "your-api-key"
    model_name: "model-name"

vision_model:
  name: "视觉识别模型"
  base_url: "https://api.example.com/v1"
  api_key: "your-api-key"
  model_name: "vision-model"
```

## 📁 数据存储结构

```
data/
├── warm/          # 热情风格数据
│   ├── characters/    # 角色文件
│   ├── groups/        # 群聊定义
│   ├── chats/         # 聊天记录
│   │   ├── single/    # 单聊记录
│   │   └── group/     # 群聊记录
│   └── images/        # 上传的图片
├── emo/           # Emo风格数据
│   └── (同warm结构)
└── dark/          # 暗黑风格数据
    └── (同warm结构)
```

### 角色文件格式
```markdown
---
model: 模型名称
---

这里是角色的人设描述...

---
## 历史记忆

这里是角色的历史记忆...
```

### 聊天记录格式
```markdown
---
type: single/group
name: 角色名/群名
characters: 角色1,角色2
---

**你** (2024-01-01 12:00:00)
你好

**角色名** (2024-01-01 12:00:05)
你好！很高兴认识你！
```
